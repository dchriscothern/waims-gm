# WAIMS-GM MVP

## Goal
A GM-facing decision support app with a premium editorial front-office aesthetic.

## MVP features
1. Local FastAPI app runs
2. Supabase auth token can be retrieved
3. One protected API route works
4. One GM dashboard page works
5. One player summary card works
6. One table/list of players or readiness notes works

## Non-goals for MVP
- no full scouting suite
- no full analytics engine
- no role-based permissions
- no multi-page frontend
- no deployment optimization yet

## Core idea
This is a front-office briefing tool, not a generic athlete dashboard.
